O Java possui oito tipos primitivos, sendo seis para valores numéricos e dois para valores textuais. 

Dos seis tipos para valores numéricos, quatro são para valores inteiros: byte, short, int e long. A diferença entre eles é apenas o tamanho: o byte tem 1 byte, o short tem 2 bytes, o int tem 4 e o long tem 8. Ou seja, cada um tem o dobro de capacidade de armazenamento do anterior. O byte comporta um espaço entre o -128 até o +127, o short armazena +- 32.767, com o negativo sempre sendo um número a mais e o positivo um a menos, pelo zero estar do lado dos positivos.
Dos seis tipos para valores numéricos, dois são para valores reais: float e double, com o último sendo o maior: o dobro do primeiro. O float tem uma capacidade de 4 bytes (até sete casas decimais) e o double tem 8 bytes (até 15 casas decimais). 1 byte = 8 bits.

Dos dois tipos textuais, temos char e boolean, com o último sempre contendo apenas "false" e "true" (outras linguagens podem converter valores como 0 e 1 para false e true em verificações, mas o Java trabalha apenas com essas duas palavras reservadas). O char é caracterizado por aspas simples na atribuição: "a", ' ', '?', '1' (sim, o char é alfanumérico), um conjunto de caracteres forma uma palavra e um conjunto de palavras forma uma frase.

Atribuir um tipo "menor" a uma variável de um tipo "maior" geralmente acontece sem problemas, mas o contrário não é possível.
